LOFI SCOUT - Artist Discovery & Labeling App
=============================================

CONCEPT
-------
Tinder-style web app for labeling artists as "Lofi vibe" or "not Lofi vibe".
Creates labeled training data to improve the Isolation Forest anomaly detection model.

RECOMMENDED STACK
-----------------
- Folder: lofi-scout/
- Backend: FastAPI (Python, consistent with existing stack)
- Frontend: Single-page HTML + vanilla JS (no build tooling)
- Storage: SQLite (swipe decisions + session tracking)
- Data sources: features.csv, PartyflockArtistItem.jsonl, LastFMSnapshot.jsonl, Chartmetric API

ARTIST CARD CONTENTS
--------------------
- Artist name + photo (Chartmetric)
- Last.fm: listeners, playcount, top tags, similar artists
- Partyflock: fans, genres, past/upcoming performances
- Chartmetric: Spotify followers, playlist reach, chart positions
- Co-appearance score with known Lofi seed artists

SWIPE LOGIC
-----------
- Swipe right / press →  =  "Lofi vibe"      → label 1
- Swipe left  / press ←  =  "not Lofi vibe"  → label 0
- Swipe up    / press ↑  =  "maybe / unsure" → label -1 (skip for now)

OUTPUT
------
- SQLite DB: decisions table (artist, label, reviewer, timestamp)
- Export: labeled_artists.csv → feeds Isolation Forest training
